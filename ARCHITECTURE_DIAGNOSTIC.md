## 1. рЃљрЃдрЃЏрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃћрЃџрЃў рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

### рЃарЃљ рЃљрЃарЃўрЃА MedGzuri

рЃЦрЃљрЃарЃЌрЃБрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃАрЃљрЃЏрЃћрЃЊрЃўрЃфрЃўрЃюрЃЮ рЃЎрЃЋрЃџрЃћрЃЋрЃћрЃЉрЃўрЃА рЃюрЃљрЃЋрЃўрЃњрЃљрЃфрЃўрЃўрЃА рЃАрЃћрЃарЃЋрЃўрЃАрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃърЃљрЃфрЃўрЃћрЃюрЃбрЃћрЃЉрЃА рЃАрЃљрЃЏрЃћрЃЊрЃўрЃфрЃўрЃюрЃЮ рЃџрЃўрЃбрЃћрЃарЃљрЃбрЃБрЃарЃўрЃА рЃЏрЃЮрЃФрЃўрЃћрЃЉрЃљрЃерЃў, рЃАрЃўрЃЏрЃърЃбрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃАрЃљ рЃЊрЃљ рЃЎрЃџрЃўрЃюрЃўрЃЎрЃћрЃЉрЃўрЃА рЃърЃЮрЃЋрЃюрЃљрЃерЃў.

### рЃарЃљрЃА рЃюрЃљрЃ«рЃљрЃЋрЃЌ рЃљрЃЏ рЃљрЃюрЃњрЃљрЃарЃўрЃерЃерЃў

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃЊрЃљ рЃърЃўрЃарЃЋрЃљрЃюрЃЊрЃћрЃџрЃў "рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃАрЃљрЃўрЃбрЃў + Vercel serverless" рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃЊрЃљрЃю **рЃАрЃљрЃЏрЃАрЃљрЃцрЃћрЃ«рЃБрЃарЃўрЃљрЃю рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃЊ**:

|------|-----------|-------------|

| **Modern Frontend** | Next.js 16 + React 19 + TypeScript | рЃљрЃ«рЃљрЃџрЃў, Railway? |
| **Legacy API** | Node.js serverless (Vercel) | рЃљрЃЦрЃбрЃўрЃБрЃарЃў |
